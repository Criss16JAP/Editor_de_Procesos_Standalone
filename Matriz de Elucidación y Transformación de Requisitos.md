# PROCESO DE CICLO DE VIDA DE SOFTWARE (ISO/IEC/IEEE 12207:2017)
## 6.4.3 PROCESO DE DEFINICIÓN DE REQUISITOS DEL SISTEMA
### DOCUMENTO 1: MATRIZ DE ELUCIDACIÓN Y TRANSFORMACIÓN DE REQUISITOS (OUTCOME A)

Este documento demuestra el cumplimiento del primer resultado del proceso de la ISO 12207, el cual exige que las necesidades, expectativas y escenarios operacionales de los usuarios finales (diseñadores de procesos, analistas, consultores, docentes) se transformen formalmente en un conjunto definido de requisitos técnicos del sistema.
